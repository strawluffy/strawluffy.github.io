命令

新建文章

hexo new 'post_name'

命令将会在目录 source/_posts/post_name.md 建立一个新的文章，默认里面的信息包括下面的内容

新建page

hexo new page 'page_name'

命令将会在目录 source/ 建立一个新的文件夹 ‘page_name’，并在这个文件里面建立一个 index.md 的文件。

页面修改

添加目录

我们新建一个page后，就可以在页面上添加一个菜单项.
查找渲染顺序

  1. 查找主题里面的 _config.yml 文件

  2. 从里面查找 menu 项目，里面是页面显示内容(如:about)和链接(如’/user’)的对应关系.

  3. 在 source 目录中查找 第二步骤中的链接目录,找到后进入第四步骤

  4. 根据语言信息显示第二步骤中的目录名称。

在页面显示的时候，会在每个目录名称前面加上一个图标，我们可以选择是否显示这个图标，默认情况下是显示的，如果不显示
需要修改主题目录中的 _config.yml 文件下面的内容，修改为 false

我们还可以定制这个显示的图标，我们首先打开链接，从里面找我们需要的图标，之后将下面的名称，填写到上面的menu选择中。如下所示, 我们设置一个地图的页面，从里面找到一个地图的图标来标识 。
